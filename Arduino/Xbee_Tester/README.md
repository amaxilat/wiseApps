 
## Xbee_Tester

Arduino application that reads the address of an Xbee Device and prints it in an LCD Screen.

### Circuit


    +-------------+
    + LCD  SHIELD +
    +-------------+
    + Xbee SHIELD +
    +-------------+
    +   ARDUINO   +
    +-------------+
